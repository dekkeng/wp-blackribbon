== Description ==
Put the black ribbon image to your wordpress website.
You can config the location of the black ribbon in the plugin setting.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/bbblackribbon` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the \'Plugins\' screen in WordPress
1. Use the Settings->WP Black Ribbon screen to configure the plugin
